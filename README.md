Yii2 Pagination
===================




Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

`
php composer.phar require --prefer-dist "c006/yii2-pagination" "*"
`

or add

`
"c006/yii2-pagination": "*"
`

to the require section of your `composer.json` file.



Composer.json
------------

>
    "repositories": [
        {
          "type": "vcs",
          "url": "https://github.com/c006/yii2-pagination.git"
        }
      ]
